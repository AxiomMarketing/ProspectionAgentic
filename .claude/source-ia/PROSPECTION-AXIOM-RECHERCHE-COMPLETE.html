Découverte & Qualification Avancée

Appel découverte (30 min Zoom)

Structure :

  • 0-2 min : Small talk
  • 2-5 min : Contexte (signal qui a déclenché le contact)
  • 5-20 min : Questions SPIN (écouter 60% du temps)
  • 20-28 min : Votre approche (comment vous résoudriez)
  • 28-30 min : Next step (audit gratuit, démo)

Questions SPIN

S (Situation) : "Vous utilisez actuellement [X] ?"
P (Problème) : "Quels challenges voyez-vous avec l'approche actuelle ?"
I (Implication) : "Et ça affecte quoi ? Revenue, temps, coûts ?"
N (Need-Payoff) : "Serait-il utile si on pouvait résoudre ça en 3 mois ?"

Audit gratuit

C'est le différenciateur Axiom : audit gratuit en 24h, sans engagement.

Contenu de l'audit :

  • Score Lighthouse (performance, accessibilité, SEO, bonnes pratiques)
  • Analyse Core Web Vitals
  • Scan RGAA (si collectivité)
  • Analyse stack technique (BuiltWith/Wappalyzer)
  • 3-5 recommandations priorisées
  • Estimation ROI si modernisation

Proposition & Closing

Post-Vente & Referral

Programme de referral

Les referrals sont le canal le moins cher (50-150 EUR CAC) et le plus qualifié (4x conversion).

Qui peut referer ?

  • Clients satisfaits (best source)
  • Agences marketing/SEO/growth (partenaires non-concurrents)
  • Comptables, avocats (leurs clients = entreprises web-ready)
  • Consultants métier (fintech, e-commerce, retail)
  • Freelances du réseau

Modèles de commission :

  • Modèle 1 : 10-15% sur premier projet (client 10K = 1-1.5K pour le référent)
  • Modèle 2 : Montant fixe (500 EUR par client qui signe)
  • Modèle 3 : Échange de services (barter)

Résultat attendu : 3-5 partenaires actifs = 5-15 refs/mois = 1-3 clients/mois

Partenariats white-label

Données choquantes (2025) : Agences outsourcing 40-60% de la prestation croissent 2.3x plus vite

Partenaires cibles pour Axiom :

  • Agences growth marketing (Growth Room, Scalability)
  • Agences SEO/SEM sans capacité dev
  • Consultants growth hacking
  • Cabinets conseil digital

Les 7 agents spécialisés

Agent 1 — Veilleur (Détection & Collecte)

Mission : Détecter en continu les opportunités de prospection

Sous-agents :

  • 1a. Veilleur LinkedIn : Monitore changements de poste, recrutements, posts pertinents
  • 1b. Veilleur Marchés Publics : Scrape BOAMP, France Marchés, filtre par mots-clés IT
  • 1c. Veilleur Signaux Web : Détecte sites lents, non conformes RGAA, tech obsolète
  • 1d. Veilleur Job Boards : Détecte offres "dev web" = besoin non satisfait

Input : APIs LinkedIn, BOAMP, BuiltWith, job boards

Output : Liste de leads bruts avec signal détecté

Fréquence : Quotidien (marchés), continu (signaux LinkedIn)

Outils : n8n, Apollo API, Warmly, scraping custom

Agent 2 — Enrichisseur (Données & Context)

Mission : Transformer chaque lead brut en fiche prospect complète

Sous-agents :

  • 2a. Enrichisseur Contact : Trouve email, téléphone, profil LinkedIn
  • 2b. Enrichisseur Entreprise : CA, effectif, secteur, stack tech, actualités
  • 2c. Enrichisseur Intent : Signaux d'intention (visites site, engagement LinkedIn, hiring)

Input : Lead brut du Veilleur

Output : Fiche prospect enrichie (15+ champs)

Outils : Apollo.io API, Clay API, Hunter.io, BuiltWith, Clearbit

Agent 3 — Scoreur (Qualification & Priorité)

Mission : Attribuer un score à chaque prospect et prioriser

Logique :

Score = ICP_match(30) + Signal_intent(25) + Engagement(15)
      + Visit_site(20) + Budget_estime(10)

Si score > 70 → HOT → envoyer au RÉDACTEUR immédiatement
Si score 40-70 → WARM → séquence multicanale standard
Si score < 40 → COLD → nurturing long terme

Input : Fiche enrichie du ENRICHISSEUR

Output : Prospect score + catégorie (HOT/WARM/COLD)

Outils : Règles custom dans n8n, ou ML simple (régression logistique)

Agent 4 — Rédacteur (Personnalisation & Outreach)

Mission : Rédiger le premier contact personnalisé (email + message LinkedIn)

Sous-agents :

  • 4a. Rédacteur Email : Draft email personnalisé (framework Challenger Sale)
  • 4b. Rédacteur LinkedIn : Draft note connexion + message post-connexion
  • 4c. Rédacteur Vidéo : Script pour vidéo Loom personnalisée (si prospect HOT)
  • 4d. Rédacteur Marché Public : Pré-analyse AO, recommandation GO/NO-GO

Input : Fiche prospect scorée + template adapté au segment

Output : Draft prêt à envoyer (ou envoyé automatiquement si WARM)

Outils : Claude API, templates paramétrés, La Growth Machine

Agent 5 — Suiveur (Sequences & Relances)

Mission : Gérer les séquences multicanales et les relances automatiques

Logique :

J0 : LinkedIn connexion (AGENT 4 draft)
J2 : Email (si pas de réponse LinkedIn)
J5 : Engagement LinkedIn (liker 2-3 posts)
J8 : 2e email (angle différent)
J14 : Dernière relance LinkedIn
J21 : Break-up message (dernier essai)

Input : Prospect en séquence active

Output : Suivi du statut (pas de réponse / réponse / RDV / refusé)

Escalade : Si réponse positive → notifier Jonathan pour prise en charge humaine

Outils : La Growth Machine, Apollo sequences, Pipedrive automation

Agent 6 — Nurtureur (Engagement Long Terme)

Mission : Maintenir le contact avec les prospects WARM et COLD

Actions :

  • Email mensuel : contenu pertinent (blog, case study, insight)
  • Engagement LinkedIn : liker/commenter les posts du prospect
  • Re-scoring trimestriel : vérifier si signaux ont changé
  • Déclenchement : si signal détecté → repasser en HOT

Input : Prospects WARM/COLD non convertis

Output : Prospects "réchauffés" remontés en priorité

Outils : ActiveCampaign/HubSpot email, n8n triggers

Agent 7 — Analyste (Reporting & Optimisation)

Mission : Mesurer la performance de chaque étape et recommander des ajustements

Métriques suivies :

  • Taux de réponse par canal et par template
  • Conversion par étape du funnel
  • CAC par segment et par canal
  • ROI par campagne
  • Pipeline value et prévisions

Output : Dashboard hebdomadaire + recommandations d'optimisation

Outils : Pipedrive dashboard, Google Sheets, n8n aggregation
